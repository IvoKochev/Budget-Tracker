Re:Little Slow to Load Pages
Thank you for the feedback. Slowness on the site could be caused by a few factors. We are currently looking into improving client side loading of the page. One of the problems however is that secure sites are always going to take more time than non-secure due to the encryption of data being sent. We've tested this and there is a big difference in load times but feel it's necessary to keep everything secure. Here are a few things you can try to speed up the page load times:

  • If you're using Internet Explorer 6, since our site is very javascript heavy, the IE6 rendering engine doesn't handle javasript as quickly as Firefox. Try using firefox to see if your load times increase.
  • In the upper right menu, there is a "themes" link. If you change the theme to simple mode, this removes the left and right panels which requires less data to load and should produce faster load times.

    We'll keep you posted on any changes we make to improve site performance.
